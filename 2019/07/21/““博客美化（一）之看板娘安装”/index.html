博客美化（一）

近期笔者在网上看到了许多大佬关于如何美化博客的文章，其中大多都很详细，其实我只要贴上大佬的地址，相信你们通过看大佬的博客也可以有很大的收获，不过我还是打算分享下其中的一部分，即左边这个小正太究竟是怎么来的。

首先我们需要安装hexo-helper-live2d，在blog根目录使用git-bash输入以下代码

npm install --save hexo-helper-live2d

接下来是配置，找到与source文件夹同一级别下的_config.yml文件，添加以下代码

# 看板娘
live2d:
  enable: true #是否允许出现
  scriptFrom: local
  pluginRootPath: live2dw/
  pluginJsPath: lib/
  pluginModelPath: assets/
  model:
  #模型名称
    use: #live2d-widget-model-chitose #西装男
         #ive2d-widget-model-epsilon2_1 #loli
         #live2d-widget-model-gf  
         ##live2d-widget-model-haru\01 
         ##live2d-widget-model-haru\02 
         live2d-widget-model-haruto    #小正太
         #live2d-widget-model-hibiki #学生妹
         #live2d-widget-model-hijiki #黑猫
         #live2d-widget-model-izumi  #女（红衣）
         #live2d-widget-model-miku
         #live2d-widget-model-ni-j #科学小正太
         #live2d-widget-model-nico #狐狸小正太
         #live2d-widget-model-nietzsche  #双马尾小loli1
         #live2d-widget-model-nipsilon #双马尾小loli2
         #live2d-widget-model-nito #菠萝头小loli
         #live2d-widget-model-shizuku 
         #live2d-widget-model-tororo  #白猫
         #live2d-widget-model-tsumiki  #绿头发的妹子
         #live2d-widget-model-unitychan  #黄发小loli
         #live2d-widget-model-wanko #碗+汪
         #live2d-widget-model-z16  #军人cos
    scale: 1
    hHeadPos: 0.5
    vHeadPos: 0.618
  display:
    superSample: 2
    width: 200
    height: 300
    position: left # 左侧还是右侧
    hOffset: 0
    vOffset: -32 # 离底部距离
  mobile:
    show: false # 手机客户端是否可见，推荐不可见，不然会影响阅读
    scale: 0.5
  react:
    opacityDefault: 0.7
    opacityOnHover: 0.2

到目前为止，你只需要到GitHub上提取模型即可，上面代码中笔者已经整合了目前为止所有的模型名称，只需要将不用的模型前加上#而需要的去掉#即可；这里还想提醒下两种安装模型包的方法。
第一种 使用git代码提取，以下为示例

npm install 模型名称

第二种则是在GitHub上直接下载解压包解压到相应位置，即blog根目录下的node_modules文件夹里。

之后运行

hexo g -d

即可完成
