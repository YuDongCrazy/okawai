Pr学习日志（一）之拉镜

发表于 2019-08-25 更新于 2019-08-26 分类于 技术分享 阅读次数：

拉镜是什么

直接描述的话笔者感觉不是很好理解，拉镜其实就是在视觉上通过镜像和运动模糊让两个不同的镜头在切换的时候制作一种“被人拉着切换”的感觉。具体可以通过观看这个视频来体会。

拉镜的制作需要什么

①Adobe Pr CC软件，具体2017 2018 2019版应该都可以（具体的软件下载，如果是壕可以考虑去购买正版Adobe软件，也可自己自行下载白嫖版）
②动态拼接等类似的预设，这里笔者是从B站What-Uncle那得到的，文件会在文章的最底部用网盘分享出来。

如何导入预设

首先找到你所下载的预设文件

在Pr里面找到“效果”界面

右键找到导入预设，再找到刚才的预设文件，点击确定即可导入成功。

拉镜的详细教程

①首先准备两端素材，在这里笔者使用的是『DRALING IN THE FRANXX』第一集内容，将内容剪取成两段相同长度或后一段长度是前一段的整数倍，这样做的原因是为了保证在拉镜的时候的流畅度。

或者也可以后面一段长一倍，前面一段长一倍

②接着将动态拼接的预设添加到素材上，这里笔者建议使用调整图层来操作，因为这样可以在不改变原视频素材的情况下实现且可以后期制作更多的效果，方便。不过这里需要强调一点就是调整图层只能统一长度，不可以像素材那样有不同的长度，其实素材之所以可以长一点也是基于使用调整图层的前提，如果真的只使用素材操作，那么仍然需要每段素材保持统一长度。

而如果使用的是调整图层，则可以按以下图示操作

③在添加完动态拼接的预设之后可以在“效果控件”进行操作，图中用小方框圈出的就是动态拼接的预设，这里我们只使用“变换”，具体我们只使用里面的位置，缩放，和快门角度三个参数。


而其中具体什么是快门角度，这里不多加赘述，你们可以通俗的理解为就是动态模糊，所以这里“使用的合成快门角度”不需要打勾，下面的具体角度可以调整为0-360度，一般越大动态模糊越明显，这里笔者设置的是200。
④笔者感觉到目前为止已经成功了一半了，剩下的就是设置关键帧以及添加缓入缓出，这里我们打算制作的效果是镜头向右拉动。首先添加位置关键帧。

这里的两个关键帧一个是默认画面参数，也就是960 540，这样的关键帧就是相当于把画面稳定在了默认阶段，当然由于额外添加了末尾关键帧，所以画面出现了动态模糊的效果，但位置仍然还是默认的状态。

之后我们设置的末尾关键帧就是将画面向右平移

可以看到由于加入了动态拼接的预设原本黑屏的画面变为了镜像。
这里我们再添加缓入缓出效果，具体操作是圈选所有的两个关键帧，然后右键，再临时差值里面找到缓入缓出，最后你会看到关键帧变成了以下样子。

这时我们再调出缓入缓出的曲线


将曲线调整为如图所示

最后将前后关键帧都拉至两端

这样我们的第一段素材就修改完毕，后一段素材基本操作不变，只不过是位置参数从左边向原始设置960 540变动而已。

这样我们的拉镜基本完成，至于缩放同理，可以从200(默认参数)变为250，然后同样修改曲线，第二段从250变为200即可。
